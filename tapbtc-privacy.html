TapBTC

Privacy Policy — Last updated May 28, 2025

TapBTC (“the app”) is developed by Ryan Evans. This page explains what information the app accesses and how it is used.

Information We Collect

TapBTC does not collect, store, or transmit any personal information. The app does not require an account, login, or any form of registration.

Network Requests

The app makes outbound requests to the following public APIs solely to display current Bitcoin price data:

  • Coinbase API — to fetch the current BTC/USD spot price
  • CoinGecko API — to fetch 24-hour high/low, all-time high, and related market data
  • Blockstream API — to fetch the current Bitcoin block height

These requests contain no personal identifiers. No data from these requests is stored beyond the current device session. No data is sent to servers operated by Ryan Evans.

Local Storage

The most recently fetched Bitcoin price is saved to the device using App Groups shared storage so that the iPhone app, Apple Watch app, and watch complications can all display a consistent price without redundant network calls. This data never leaves your device and is overwritten each time a new price is fetched.

Analytics and Advertising

TapBTC contains no analytics SDKs, no advertising SDKs, and no third-party tracking of any kind.

Children

TapBTC does not knowingly collect information from children under 13. Because no personal information is collected from anyone, the app is safe for all ages.

Changes to This Policy

If the privacy practices of TapBTC change, this page will be updated and the “last updated” date at the top will reflect the change.
